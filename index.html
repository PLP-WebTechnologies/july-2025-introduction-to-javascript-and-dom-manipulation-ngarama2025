Part 1: Variables and Conditionals

JavaScript variables are declared using let and const.

// Variable declarations and conditionals
let score = 85;
const passingGrade = 60;
let message = "";

if (score >= passingGrade) {
  message = "You passed the exam!";
} else {
  message = "You need to try again.";
}

Part 2: Custom Functions

Functions are reusable blocks of code that perform a specific task.

// Function to calculate area of a rectangle
function calculateArea(width, height) {
  return width * height;
}

// Function to check if a number is even
const isEven = function(num) {
  return num % 2 === 0;
};

Part 3: Loops

Loops are used to execute the same block of code multiple times.

// For loop example
for (let i = 1; i <= 5; i++) {
  console.log("Count: " + i);
}

// While loop example
let count = 1;
while (count <= 5) {
  console.log("While count: " + count);
  count++;
}

Part 4: DOM Interactions

The Document Object Model (DOM) allows JavaScript to interact with HTML elements.

// Changing text content
document.getElementById('elementId').textContent = 'New text';

// Handling click events
button.addEventListener('click', function() {
  // Code to execute on click
});

// Changing styles
element.style.backgroundColor = 'blue';
